<h3>源码简介：</h3>
<p>emlog远程图片本地化插件 支持Emlog Pro版本。此插件会把文章内所有本地链接的图片进行本地化，以防小白在不懂得情况下开启致使文章发布缓慢，所以默认不勾选勾选后将会本地化非本地连接的图片，本地化图片资源不入资源管理库，远程图片体积越大，本地化的速度就会越慢顾名思义，当一篇文章内非本地图片链接过多的时候发布文章时本地化处理缓慢！<p>
